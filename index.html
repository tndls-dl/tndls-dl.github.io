🌿 Ingrevia 챗봇

화장품 전성분 기반 맞춤형 추천 에이전트 개발

분석 목표: 화장품 전성분 데이터 분석을 통해 고객의 피부 고민 및 알레르기 유발 성분을 고려하여, 개인 맞춤형 화장품을 추천하고 성분에 대한 궁금증을 해소하는 지능형 챗봇 서비스를 구축했습니다.

핵심 기술 및 아키텍처: LLM Agent 구조를 기반으로 설계하여, 사용자의 질문 의도를 파악하고 내부 데이터베이스(전성분 데이터)에서 정보를 검색하여 답변하는 RAG(검색 증강 생성) 시스템을 구현했습니다. RAG와 Agent라는 최신 AI 기술을 활용하여 LLM의 한계를 극복하고 정확한 정보를 제공합니다.

데이터 처리: 대량의 화장품 전성분 데이터를 수집 및 전처리하고, 이를 벡터 데이터베이스 형태로 변환했습니다. 성분-피부 타입-효능 간의 관계를 분석하여 추천 로직의 기반을 마련했습니다.

시스템 구현: FastAPI를 활용하여 API 서버를 구축하고, 챗봇 인터페이스와 LLM Agent 시스템을 연결하여 사용자가 실시간으로 서비스를 이용할 수 있도록 배포 환경까지 구현했습니다.

비즈니스 기여: 소비자들이 복잡한 화장품 성분을 쉽게 이해하고 피부 민감도를 고려한 안전한 소비를 할 수 있도록 지원하며, 구매 전환율과 고객 만족도 향상에 기여할 수 있는 솔루션을 제시했습니다.

🎬 한국 영화 데이터 분석

한국 영화 흥행 요인 분석 및 산업 구조 탐색 (2005-2019)

분석 목표: 2005년부터 2019년까지 한국 영화 데이터를 분석하여 흥행에 영향을 미치는 주요 요인을 규명하고, 영화 산업 전반의 구조적 인사이트를 도출했습니다. 코로나19 특수성을 배제하기 위해 분석 기간을 한정했습니다.

데이터 수집 및 통합: TMDB, KOFIC(관객/수익), 네이버 영화 크롤링(평점)의 다중 소스 통합 전략을 사용하고, 수동 입력으로 약 900편의 최종 데이터셋 품질을 보강했습니다. 장르/감독/배우 텍스트 데이터를 파싱하여 리스트로 변환하고, 개봉월/계절, 런타임 구간 등 흥행 분석을 위한 핵심 파생 변수를 생성했습니다.

주요 분석 결과:

1) 흥행 공식 존재: 장르별로 개봉 시기 특징이 다르며(사극/액션: 여름/명절 집중, 공포: 여름 ROI 극대화), 장르별 리스크-리턴 편차(사극/재난: 고수익/고위험, 드라마/코미디: 안정적)를 확인했습니다.

2) 평점과 흥행의 괴리: 평점과 흥행 수익 간의 상관계수는 약 0.2로 약한 상관관계를 보였으며, 저평점·고흥행 그룹이 대기업 배급사(CJENM, 쇼박스 등) 중심의 '규모의 힘'으로 흥행했음을 규명했습니다.

3) 네트워크 효과: 상위 40개 감독/배우 콤보를 기준으로 협업 네트워크를 시각화하여, 흥행이 개별 스타가 아닌 반복적 협업 구조(브랜드 효과)에서 비롯됨을 확인했습니다.

향후 계획: 예산 데이터 부재로 ROI 분석이 불가했던 구조적 한계를 인식하고, 향후 ROI 추정 모델 개발 및 OTT 데이터 결합을 통한 통합 성과 분석을 계획하고 있습니다.
